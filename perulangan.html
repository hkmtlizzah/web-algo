Perulangan (Looping)

Perintah Looping FOR

Merupakan struktur perulangan yang digunakan untuk mengulangi suatu proses yang telah diketahui jumlah perulangannya (Counted Loop). Pernyataan FOR digunakan untuk melakukan looping. Pada umumnya looping yang dilakukan oleh FOR telah diketahui batas awal, syarat looping, dan perubahannya. Selama kondisi terpenuhi, pernyataan akan terus dieksekusi.

Bentuk umum perulangan FOR adalah sebagai berikut:

Keterangan:

• Ungkapan1 atau inisialisasi: digunakan untuk memberikan inisialisasi terhadap variabel pengendali loop.

• Ungkapan2 atau syarat perulangan: digunakan sebagai pemegang kontrol terhadap pengulangan. Bagian ini akan menentukan suatu kondisi perulangan untuk diteruskan atau dihentikan.

• Ungkapan3 atau pengubah nilai pencacah: digunakan sebagai pengatur perubahan nilai variabel pengendali looping.


Contoh sintaks pengulangan FOR adalah sebagai berikut:


• Pernyataan Int i=0 merupakan inisialisasi dalam sintaks perulangan for. Sintaks ini berarti variabel i memiliki nilai awal 0 dan memiliki jenis data integer.

• Pernyataan i<5 merupakan syarat pengulangan. Perulangan akan terus dilakukan apabila variabel i memiliki nilai < 5.

• Pernyataan i++ merupakan pengubah nilai pencacah.


Perintah Looping WHILE

Merupakan jenis perulangan yang digunakan apabila jumlah perulangan belum diketahui (Uncounted Loop). Perulangan ini akan terus berlanjut selama kondisinya bernilai benar dan akan berhenti bila kondisinya bernilai salah.

Bentuk umum perulangan WHILE adalah sebagai berikut:

Contoh Soal

• Buatlah algoritma menggunakan flowchart dan pseudocode untuk menampilkan baris yang berisi tulisan “Selamat Pagi” sesuai dengan keinginan user!


Algoritma:

1. Masukkan Bil

2. Ulang hingga mencapai nilai Bil

3. Tampilkan “Selamat Pagi”

4. Bil = Bil + 1 (pencacah naik)

5. Akhir-ulang


Pseudocode:

Input Bil

For (i=1; i<= bil; i++)

“Selamat Pagi”

i = i + 1

End for

• Buatlah algoritma menggunakan flowchart dan pseudocode untuk menampilkan bilangan ganjil di antara 1 sampai dengan 10!


Algoritma:

1. Masukkan Bil

2. Ulang selama Bil <= 10

3. Tampilkan Bil

4. Bil = Bil + 2

5. Akhir-ulang


Pseudocode:

Bil = 1

While (Bil <= 10)

Tampilkan (Bil)

Bil = Bil + 2

End While
