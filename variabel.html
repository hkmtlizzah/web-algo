Variabel dan Operator

Variabel dan Konstanta

Variabel adalah suatu nama yang menyatakan tempat dalam memori untuk menyimpan suatu nilai dan nilainya dapat diubah sewaktu-waktu ketika dibutuhkan.

Sebagai contoh variabel dalam kehidupan sehari-hari adalah ketika memasak, kita akan membutuhkan panci (variabel) untuk memasak. Isi dari panci itu sendiri bisa berubah-ubah ketika dibutuhkan. Begitu juga di algoritma atau bahasa pemrograman. Sebelum memproses suatu masalah menggunakan bahasa pemrograman, kita harus mendeklarasikan variabel dan tipe datanya terlebih dahulu untuk membantu memecahkan masalah tersebut.


Konstanta adalah suatu data yang digunakan dalam pemrograman dan sifatnya tetap.

Sama halnya dengan variabel, fungsi dari konstanta adalah sebagai tempat penyimpanan sementara di dalam memori, hanya saja nilai yang dimiliki oleh konstanta tidak dapat diubah. Konstanta harus di deklarasikan terlebih dahulu pada awal program.

Tipe Data

Dalam bahasa pemrograman, terdapat 5 jenis tipe data yang biasa digunakan. Masing-masing tipe data memiliki fungsinya masing-masing, seperti berikut:

  1. Integer

    2. Merupakan tipe data yang menyatakan bilangan bulat antara -2147483647 sampai dengan +2147483647. Penggunaan tipe data integer biasanya untuk data yang akan melakukan operasi matematika seperti penjumlahan, pengurangan, perkalian, dan pembagian.

  2. Float

    3. Merupakan tipe data yang menyatakan bilangan real atau pecahan dengan ketelitian rendah (6-7 digit). Dapat menampung bilangan antara 10-38 sampai dengan 1038. Untuk mendapatkan hasil yang lebih baik dalam proses pengoperasian angka, maka dapat digunakan tipe data ini.

  3. Char

    4. Merupakan tipe data yang menyatakan sebuah karakter. Karakter tersebut bisa berupa simbol, alphabet, atau angka.

  4. String

    5. Merupakan tipe data khusus untuk string yaitu data yang berisi kalimat atau kata.

  5. Boolean

    6. Merupakan tipe data untuk menyatakan nilai logika true atau false

Operator

Operator adalah simbol yang digunakan dalam pemrograman dan memiliki fungsi untuk menyusun suatu ekspresi. Ekspresi merupakan suatu bentuk yang digunakan unruk menghasilkan sebuah nilai.

    Jenis Operator Berdasarkan Kegunaannya:

  1. Operator Aritmatika

    2. Operator aritmatika digunakan untuk melakukan perhitungan aritmatika. Daftar operator matematika dapat dilihat pada tabel dibawah ini

  2. Operator Logika

    3. Operator logika digunakan untuk menghubungkan dua buah ekspresi pembandingan menjadi sebuah ekspresi kondisi. Daftar operator logika dapat dilihat pada tabel dibawah ini

  3. Operator Relasional

    4. Operator relasional atau operator pembanding adalah operator yang digunakan untuk membandingkan dua buah nilai. Operator relasional akan menghasilkan kondisi BENAR dan SALAH. Operator relasional hanya dapat digunakan untuk tipe dasar. Daftar operator relasional dapat dilihat pada tabel dibawah ini.
