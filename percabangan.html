Percabangan

Pengertian Percabangan

Percabangan adalah istilah yang digunakan untuk menggambarkan alur program yang bercabang. Pada flowchart, logika “jika…maka” digambarkan dalam bentuk cabang. Selain percabangan, struktur ini juga disebut: control flow, decision, struktur kondisi, dsb. Dengan menggunakan percabangan, memungkinkan program untuk mengambil tindakan sesuai dengan kondisi yang telah diberikan.

Dalam percabangan, keputusan yang diambil oleh program akan diambil berdasarkan ekspresi kondisi yang diberikan. Ekspresi kondisi adalah pernyataan program yang menyatakan pertanyaan benar atau salah mengenai suatu data yang ada pada kode program.

Jenis Percabangan

1. Untuk memilih 2 kondisi

Statement IF-ELSE mempunyai pengertian “Apabila kondisi bernilai benar, maka pernyataan1 akan dikerjakan. Apabila kondisi bernilai salah, maka pernyataan2 akan dikerjakan”. Diagram alir dan penulisan dari penyataan IF-ELSE dapat dilihat pada gambar dibawah

2. Untuk memilih lebih dari 2 kondisi

Memiliki pengertian yang sama dengan statement IF-ELSE, bedanya ialah pada IF-ELSE bertingkat terdapat pernyataan di dalam pernyataan lain sehingga memiliki lebih dari 2 blok pilihan. Diagram alir dan penulisan dari penyataan IF-ELSE bertingkat dapat dilihat pada gambar dibawah.

Contoh Penggunaan


•Buatlah algoritma menggunakan flowchart dan pseudocode untuk menentukan apakah bilangan yang diinput adalah bilangan ganjil atau bilangan genap!


Algoritma:

1. Masukkan sebuah bilangan

2. Bagi bilangan tersebut dengan angka 2

3. Jika bilangan menghasilkan sisa pembagian 0, maka genap

4. Jika bilangan tidak menghasilkan sisa pembagian 0, maka ganjil


Pseudocode:

Read Bilangan

If bil mod 2 = 0 then,

“Output Genap”

Else

“Output Ganjil”

End if


•Buatlah algoritma menggunakan flowchart dan pseudocode untuk menentukan kelulusan siswa apabila nilai lebih dari 70!


Algoritma:

1. Masukkan nama siswa dan nilai

2. Jika nilai > 70, maka keterangan lulus

3. Jika nilai < 70, maka keterangan tidak lulus

4. Tampilan nama, nilai, dan keterangan siswa


Pseudocode:

Read nama, nilai

If nilai > 70

“Lulus”

Else

“Tidak Lulus”

End if
